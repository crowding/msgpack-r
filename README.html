msgpack for R

This is a high speed [msgpack][2] encoder and decoder for R, based on the [CWPack][1] C implementation. 1: https://github.com/clwi/CWPack 2: https://msgpack.org

msgpack is a binary data format with compact encoding and data structure support similar to JSON. It can be a drop-in replacementfor JSON in most applications. It is designed to be fast to parse and compact to transmit and store.

Installation

library(devtools)


install_github("crowding/msgpack-r")

Usage

library(msgpack)


x <- packMsg( list(compact=TRUE, schema=0) )


x
##  [1] 82 a7 63 6f 6d 70 61 63 74 c3 a6 73 63 68 65 6d 61 00

str(unpackMsg( x ))
## List of 2
##  $ compact: logi TRUE
##  $ schema : int 0

Connections / Streaming

conOut <- rawConnection(raw(0), open = "w") #or socketConnection, etc


writeMsg("one", conOut)


writeMsgs(list(2, c(buckle=TRUE), c(owner="my", type="shoe")), conOut)


conIn <- msgConnection(rawConnection(rawConnectionValue(conOut), open = "r"))


readMsgs(conIn, 2)
## [[1]]
## [1] "one"
## 
## [[2]]
## [1] 2

readMsg(conIn)
## buckle 
##   TRUE

readMsgs(conIn)
## [[1]]
##  owner   type 
##   "my" "shoe"
